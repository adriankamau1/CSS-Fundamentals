Before we continue, we recommend you have basic HTML syntax knowledge

Selectors Description Examples
Type matches the type of element in HTML document HTML: <p>, <h1> Selector: p,h1
Universal Targets all elements of a page * {declaration} = all elements {declaration}
class attribute used to select an element, must use "." before class name when calling in css sheet
*multiple elements can share the same class*		 HTML: class="ex1"; Selector: .ex1{declaration}
ID attribute used to select an element, must use "#" before class name
*only one element per id*		 HTML: id="ex2"; Selector: #ex2{declaration}
